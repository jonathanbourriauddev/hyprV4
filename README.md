# 🌌 Catppuccin Hyprland Rice - CachyOS

Un environnement de travail minimaliste, performant et cohérent basé sur le thème **Catppuccin Mocha**. Entièrement optimisé pour la productivité, le développement Docker et la gestion de VM.

## 🚀 Installation en une ligne

Si vous êtes sur une installation fraîche de **CachyOS**, lancez simplement cette commande pour tout déployer (configs, thèmes, terminaux et GRUB) :

```bash
git clone [https://github.com/TON_PSEUDO/dotfiles.git](https://github.com/TON_PSEUDO/dotfiles.git) ~/dotfiles && cd ~/dotfiles && chmod +x install.sh && ./install.sh
